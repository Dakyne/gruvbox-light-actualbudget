# Gruvbox Light for Actual Budget

A light custom theme for [Actual Budget](https://actualbudget.org) based on the [Gruvbox](https://github.com/morhetz/gruvbox) color palette by Pavel Pertsev.

Warm, retro-groove tones — easy on the eyes, high readability.

## Installation

### From the catalog

Settings → *Show advanced settings* → *Experimental features* → enable **Custom themes**, then pick **Gruvbox Light** from the catalog.

### Manual

1. Enable custom themes (see above).
2. Copy the contents of [`actual.css`](./actual.css).
3. Settings → *Themes* → *Add custom theme* → paste the CSS.
